# Blog

Web app that shows a list of posts, based on the course [Modern React with Redux [2019 Update]](https://www.udemy.com/react-redux/) available on Udemy created by [Stephen Grider](https://www.udemy.com/react-redux/).

## Used technologies

- React
- Redux
- Thunk
- Axios
- Lodash
- JSONPlaceholder
- Font Awesome
- HTML
- CSS
- JavaScript
